Dictionary

Dictionaries are dynamic hashmaps that are made of different entries. Each entry is made from a key and a value.

Keys are typically strings or integer data types for easy navigation while the matching values can be any data type. Keys within a dictionary are unique where if a duplicate key is found, the new value will override the old value.

A dictionary will function as a Lookup Table where data can be referenced and edited by adding or removing keyvalue pairs and searching for existing values by its matching key. Data Dictionaries are created for easy data storage such as the Oak Ridge national Laboratory which provides basic information for different variables.
