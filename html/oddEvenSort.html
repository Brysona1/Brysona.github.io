Odd-Even Sort

An Odd-Even Sort is a variation of a Bubble Sort that utilizes the Divide & Conquer methodology. A given unsorted array becomes sorted within two phases, alternating between a Bubble Sort on the Odd indexes of the array followed the Even indexes. These iterations repeat until no more values can be swapped, indicating a completed sorted array.

An Odd-Even Sort is ideal when working with larger amounts of data. Where a regular Bubble Sort excels in uniform usage, an Odd-Even Sort Often performs better in numbers. Often, multiple processors are set to run the algorithm in simultaneous instances (parallelism) to process these multitudes of data.
