Doubly Linked List

Doubly Linked Lists define nodes that contain a previous pointer, a value, and a next pointer. At any given node, the pointers will access subsiding nodes in either direction.

This makes a Doubly Linked List more flexible than a regular Linked List. At the cost of being slower to run and using up more memory, the Doubly Linked List is easy to reverse and has a low overhead. Some implementations include webpage navigation, programming a deck of cards, running thread schedulers among other uses.
